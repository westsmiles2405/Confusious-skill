# 📜 孔子工作陪伴 — VS Code Extension

[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)
[![VS Code](https://img.shields.io/badge/VS%20Code-%5E1.85.0-007ACC?logo=visual-studio-code)](https://code.visualstudio.com/)

> *"学而时习之，不亦说乎？有朋自远方来，不亦乐乎？"*

让至圣先师 **孔子** 以文言美学与君子之道陪伴你的 VS Code 工作之旅。他会用经典文言句式鼓励你、以仁义礼智信审视你的代码、在你疲惫时以论语智慧温和抚慰，还会谈六艺、讲弟子故事，让你在编程中感悟两千五百年前的教育哲学。

---

## ✨ 功能一览

| 功能 | 命令 | 说明 |
|------|------|------|
| 📖 今日讲学 | `孔子：今日讲学` | 根据时间段生成文言开场问候，开启今日修学 |
| ⏱ 修身专注 | `孔子：开始修身专注` | 启动番茄钟（默认 25 分钟），以"修身齐家"之名专注 |
| ⏹ 结束修身 | `孔子：结束修身专注` | 手动停止当前番茄钟 |
| 💪 为我鼓劲 | `孔子：为我鼓劲` | 获得一段文言鼓励，如"天将降大任于斯人" |
| 🔍 正名审代码 | `孔子：正名审代码` | 汇总工作区错误，以"正名"思想审视代码质量 |
| 🏁 今日收工 | `孔子：今日收工` | 结束今日工作，回顾修身完成数与对话统计 |
| 🎯 谈谈六艺 | `孔子：谈谈六艺` | 听孔子讲礼、乐、射、御、书、数，映射编程实践 |
| 📚 论语一则 | `孔子：论语一则` | 随机推送一则论语名句，联系编程场景解读 |
| 👥 聊聊弟子 | `孔子：聊聊弟子` | 聊聊颜回、子路、子贡、冉有等弟子故事 |
| 🎭 切换风格 | `孔子：切换说话风格` | 在文言 / 白话 / 严师三种语言模式间切换 |

## 🏛 侧边栏面板

活动栏会出现孔子图标，点击打开 **"杏坛书院"** 聊天面板：

- 支持自由对话（智能关键词识别 + 角色化文言回复）
- 每保存 10 次文件，孔子会主动发一句工作台词
- 长时间未操作会触发温和的闲置提醒
- 展示今日统计：修身（番茄数）、对话数、保存数
- 古色古香的 **朱红 + 金色** 界面主题
- 沉稳的打字机效果，契合夫子的从容不迫

## 🎭 三种说话风格

通过 `孔子：切换说话风格` 命令随时切换：

| 模式 | 风格描述 | 适合场景 |
|------|----------|----------|
| **文言模式** (默认) | 纯文言美学，"者……也"定义式、"不亦……乎"反诘式 | 沉浸式体验，感受古典韵味 |
| **白话模式** | 半文半白，保留文言韵味但更易理解 | 日常使用，兼顾可读性 |
| **严师模式** | 直接犀利，如同夫子当面训诫弟子 | 需要压力驱动时 |

## 🧠 设计理念

本插件基于对孔子思想的深度研究报告构建，涵盖：

- **文言句式美学**：复刻"者……也"、"不亦……乎"、"……而已矣"等经典句式
- **六艺映射**：将礼（规范）、乐（优雅）、射（精准）、御（驾驭）、书（文档）、数（算法）映射到现代编程
- **论语编程解读**：每一则论语都联系实际编程实践
- **因材施教**：根据用户话题与情绪智能识别，给出不同风格的回复
- **仁义礼智信**：以五常思想审视代码质量与工程伦理
- **弟子故事**：借颜回之安贫、子路之勇决、子贡之灵通，启发不同编程风格

## ⚙️ 配置项

| 设置 | 默认值 | 说明 |
|------|--------|------|
| `kongzi.pomodoroMinutes` | 25 | 修身专注时长（分钟） |
| `kongzi.breakMinutes` | 5 | 休憩时长（分钟） |
| `kongzi.enableIdleReminder` | true | 是否在长时间未操作时提醒 |
| `kongzi.idleMinutes` | 30 | 闲置多少分钟后提醒 |
| `kongzi.enableStatusBar` | true | 是否显示状态栏入口 |

## 🚀 快速开始

### 从源码构建

```bash
git clone https://github.com/westsmiles2405/Confusious-skill.git
cd Confusious-skill
npm install
npm run compile
```

然后在 VS Code 中按 **F5** 启动 Extension Development Host 调试。

### 从 .vsix 安装

```bash
code --install-extension kongzi-companion-0.1.0.vsix
```
