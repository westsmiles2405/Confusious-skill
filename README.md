# 📜 孔子工作陪伴 (Kongzi Companion)

> 让至圣先师孔子以文言美学与君子之道陪伴你的 VS Code 工作之旅。

## 功能

| 命令 | 说明 |
|------|------|
| `孔子：今日讲学` | 开场问候，开启今日修学 |
| `孔子：开始修身专注` | 启动番茄钟（默认 25 分钟） |
| `孔子：结束修身专注` | 手动停止番茄钟 |
| `孔子：为我鼓劲` | 获得孔子的文言鼓励 |
| `孔子：正名审代码` | 让孔子审视你的代码质量 |
| `孔子：今日收工` | 结束今日工作，回顾修身完成数 |
| `孔子：谈谈六艺` | 听孔子讲礼乐射御书数 |
| `孔子：论语一则` | 随机一则论语，联系编程实践 |
| `孔子：聊聊弟子` | 聊聊颜回、子路、子贡等弟子 |
| `孔子：切换说话风格` | 在文言/白话/严师三种模式间切换 |

## 特色

- **文言美学**：以"者……也"定义式、"不亦……乎"反诘式等经典句式与你对话
- **君子之道**：用仁义礼智信的处世逻辑激励工作
- **六艺全能**：将礼、乐、射、御、书、数映射到现代编程实践
- **论语智慧**：随机推送论语名句，结合编程场景解读
- **因材施教**：根据你的情绪和话题智能切换回复风格
- **三种风格**：文言模式（纯文言）、白话模式（半白话）、严师模式（直接有压力）
- **闲置提醒**：长时间未操作时温和提醒
- **打字机效果**：沉稳的逐字输出，契合夫子的从容不迫

## 设置

| 选项 | 默认值 | 说明 |
|------|--------|------|
| `kongzi.pomodoroMinutes` | 25 | 修身专注时长 |
| `kongzi.breakMinutes` | 5 | 休憩时长 |
| `kongzi.enableIdleReminder` | true | 闲置提醒开关 |
| `kongzi.idleMinutes` | 30 | 闲置多少分钟后提醒 |
| `kongzi.enableStatusBar` | true | 状态栏显示 |

## 安装

```bash
# 从 .vsix 安装
code --install-extension kongzi-companion-0.1.0.vsix
```
